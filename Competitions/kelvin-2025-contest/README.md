# Lord Kelvin Programming Competition 2025，12/7

这个仓库用于记录我在 2025 年 Lord Kelvin Programming Competition 中的代码和相关材料。

This repository records my solution code and related materials for the 2025 Lord Kelvin Programming Competition.

## 背景 / Context

- 比赛过程中主要使用：
  - Python：比赛时我对 C++ 不熟，一开始使用的是 Python 版本；
  - C++：后期在比赛规则允许的前提下，借助 AI 助手提供的代码思路，再由我自己修改和调试，得到更高分的版本。

- 运行环境：
  - CodeGrade 平台 + 本地开发环境。

- 参赛赛道：
  - VIP

## 目录说明 / Structure

- `src-python/`  
  比赛期间实际提交或测试用的 Python 版本代码。  
  Python solutions that I wrote and used during the contest.

- `src-cpp/`  
  赛后在 AI 协助下，用 C++ 重写 / 优化的版本，由我自己理解、修改和调试。  
  Post-contest C++ versions developed with the help of an AI assistant, then understood, verified and tuned by myself.

- `docs/handbook.pdf`  
  官方提供的竞赛手册，仅作为个人学习与备查。  
  Official contest handbook, kept here for personal study and reference.

- `results/`  
  分数截图或日志，用来记录我在 CodeGrade 上的最好成绩。  
  Screenshots or logs of my best scores on CodeGrade.
